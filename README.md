# friendsforever

Simple invite-based download landing app for campaign APK/IPA downloads.

## Run locally

```bash
npm install
npm start
```

Open:

`http://localhost:3000/?invite_code=<code>`

## Environment

- `CAMPAIGN_API_BASE` (default: `https://campaign-manager-api-8zjt.onrender.com`)

## Render

Use the included `render.yaml` blueprint.

Health check endpoint: `/healthz`
